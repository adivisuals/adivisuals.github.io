# Aditya Vishwakarma — Video Editor Portfolio

Live: `https://adivisuals.github.io/`

A cinematic, single-page React portfolio (Vite + Tailwind v4) with:

- Home / Work / About / Contact pages
- Decap CMS admin panel
- EmailJS contact form (mailto fallback)
- GitHub Pages deployment via GitHub Actions
- Security hardening (CSP, validation, honeypot, noopener)

## Local development
```bash
npm install
npm run dev
